✨ Fonctionnalités

Tout ce qu'il faut pour auditer vos backups en toute confiance.

🔍

Scan approfondi

Comparez un dossier source avec son backup fichier par fichier. Détection des fichiers manquants, modifiés et corrompus.

🔐

Checksums SHA-256

Vérification bit-par-bit de l'intégrité. Deux fichiers de même taille peuvent être différents — Backcheck le détecte.

📊

Rapports détaillés

Exportez vos résultats en HTML ou PDF. Cartes de résumé, table détaillée, filtres par statut.

🔄

Sync backup

Synchronisez votre backup en un clic : copie les fichiers manquants, remplace les modifiés, supprime les orphelins.

📈

Progression temps réel

Visualisez l'avancement du scan en temps réel : phase en cours, nombre de fichiers, pourcentage, fichier traité.

📋

Historique

Retrouvez tous vos scans passés avec leurs résultats. Comparez l'évolution de vos backups dans le temps.

🌐

Bilingue FR/EN

Interface entièrement traduite en français et anglais. Votre préférence est sauvegardée automatiquement.

🛡️

100% sûr

Backcheck ne modifie jamais vos fichiers. Lecture seule garantie. Vos données ne sont jamais touchées.

📁

Multi-projets

Scannez un seul projet ou tous vos projets en une fois. Compatible avec les dossiers de 1 Go à 5 To.

📖 Comment ça marche

Trois étapes simples pour vérifier vos backups.

1

📂 Sélectionnez vos dossiers

Choisissez le dossier source (vos projets originaux) et le dossier backup (NAS, disque externe, etc.). Vous pouvez aussi coller le chemin directement.

2

🔍 Lancez le scan

Backcheck parcourt récursivement les deux dossiers, compare les fichiers par taille puis par checksum SHA-256. La progression s'affiche en temps réel.

3

📊 Analysez les résultats

Consultez le résumé : fichiers identiques, manquants, modifiés. Filtrez par statut. Exportez en HTML/PDF. Synchronisez si nécessaire.
